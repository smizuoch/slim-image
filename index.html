画像最適化ツール

画像の幅・高さと拡張子を変えずに、目標サイズ以下で最高画質の候補を探索します。処理はすべてブラウザ内だけで完結します。

Completely Static Site

GitHub Pages にそのまま載せられる画像最適化ツール

このツールは HTML / CSS / JavaScript / Rust-WASM のみで構成された完全静的サイトです。サーバーアップロード、外部 API、Node.js 本番依存はありません。

PNG / JPEG / WebP を対象に、幅・高さと拡張子を変えず、目標サイズ以下の制約内で品質スコアが最大になる候補を探索します。

主な仕様

  • ドラッグ＆ドロップとファイル選択に対応
  • Before / After プレビューと詳細ログ表示
  • 処理は Web Worker + WASM で実行
  • PNG の透過 / WebP の透過 / JPEG の形式維持
