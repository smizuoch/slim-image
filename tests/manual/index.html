manual test checklist

アプリを開く

  1. PNG / JPEG / WebP をそれぞれ読み込み、元画像情報が表示されることを確認する。
  2. 元サイズより大きい目標値を入れると「すでに条件を満たしている」と表示されることを確認する。
  3. JPEG 入力で JPEG 出力、WebP 入力で WebP 出力、PNG 入力で PNG 出力になることを確認する。
  4. 透過 PNG / WebP の背景が保持され、Before / After で縦横サイズが変わらないことを確認する。
  5. 非常に厳しい目標値を設定し、目標未達時に失敗メッセージが出ることを確認する。
  6. 処理中に詳細ログと試行候補数が更新されることを確認する。
